# acidclaw-zerobaek

**ZeroBaek (제로백): OpenClaw와 함께 0부터 100까지 자동 개발하는 프로젝트**

본 프로젝트는 고도(Godot) 엔진 대신 웹 기반(HTML/CSS/JS)으로 방향을 전환하여, OpenClaw AI 에이전트(AcidClaw)가 주도적으로 크론잡(Cron Job)을 통해 매일 스스로 코드를 작성하고, 진척도를 기록하며, Git 커밋/푸시까지 수행하는 완전 자동화 파이프라인 실험입니다.

## 기술 스택
- HTML5 Canvas / Vanilla JavaScript
- OpenClaw Agent (Cron 기반 오토메이션)

## 핵심 목표
1. **자율 주행 개발:** 에이전트가 매일 스스로 `체크리스트.md`와 `일정.md`를 확인하고 다음 마일스톤을 구현.
2. **진척도 트래킹:** 에이전트가 직접 마크다운 문서를 업데이트하며 작업 내역을 문서화.
3. **웹 시네마틱 축구:** 1인칭 시점의 리듬/온레일 축구 경험을 웹 브라우저 상에서 렌더링.
