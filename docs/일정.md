# 개발 일정표 (제로백)

에이전트(AcidClaw)는 매일 본 문서를 갱신하며 진행 상황을 기록한다.

## 마일스톤 1: 기반 시스템 구축 (웹 기반 렌더러)
- HTML Canvas 기본 구조 및 리소스 로더 세팅 (완료)
- 메인 루프 (requestAnimationFrame) 구현 (완료)

## 마일스톤 2: 1인칭 카메라 및 레일 트랙 이동 구현
- 원근 투영(Perspective Projection) 및 마우스 시야 회전(Pointer Lock) 구현 (완료)
- 온레일 트랙의 Z축 이동 스크립트 및 키보드(WASD) 이동 구현 (완료)
- 시야각 및 좌우 틸트 카메라 효과 적용 (완료)

## 마일스톤 3: 리듬/조작 메커니즘 연동
- LMB/RMB 양발 스텝/드리블 조작 시스템 (완료)
- 템포에 따른 타이밍/판정 및 콤보 UI 구현 (완료)

## 마일스톤 4: 시각적/감각적 연출 (웹 기반 렌더러 반영)
- 장애물(수비수) 데이터 구조 및 2D 렌더링 구현 (완료)
- 장애물 충돌 판정 및 패널티 로직 연동 (완료)
- 화면 모서리 황금빛 파티클/글로우 셰이더 처리 (완료)
- 시간 흐름에 따른 Color Grading (CSS Filters / Canvas Compositing) (완료)

## 마일스톤 5: 챕터별 시나리오 구성 및 엔딩
- 적재적소에 튜토리얼 및 NPC 장애물 배치 (완료)
- 엔딩 씬 (과거의 자아 조우) 구현 (완료 - 화면 필터, 크레딧, 황금빛 자아 오브젝트 임시 렌더링 추가)

## 마일스톤 6: 게임 플레이 폴리싱 및 Web Audio API 사운드 도입
- Web Audio API (Synth 기반) 발걸음 사운드(LMB/RMB) 추가 (완료)
- 장애물(수비수) 충돌 시 크래시 효과음 연동 (완료)
- 게임 내 10, 20 단위 콤보 달성 시 성공/환호성 효과음 재생 구현 (완료)

## 마일스톤 7: 게임플레이 다양성 및 시각 효과 개선
- 속도 기반 동적 FOV 변화 (속도감 줌 효과) 연동 (완료)
- 배경 3D 별(Starfield) 파티클 시스템 (완료)

## 마일스톤 8: 게임 최적화 및 UI/UX 개선
- 일시정지(Pause) 기능 추가 및 상태 연동 (완료)
- 인게임 HUD 진행 시간(Playtime) 표시 추가 (완료)

## 마일스톤 9: 모바일/터치 환경 지원 및 조작성 개선
- 터치스크린 기기를 위한 화면 좌/우 탭 조작 추가 (완료)
- 터치 드래그를 이용한 시야 회전 로직 구현 (완료)

## 마일스톤 10: 데이터 저장 및 부가 기능
- 로컬 스토리지를 활용한 최고 플레이 기록(Best Playtime) 저장 및 타이틀 화면 표시 로직 구현 (완료)

## 마일스톤 11: 코드 정리 및 최종 마무리
- 불필요한 콘솔 로그 제거 및 메인 스크립트 최적화 (완료)
- 최종 테스트 및 배포 준비 완료 (완료)

## 마일스톤 12: 시각적 폴리싱 추가
- 타이틀 화면에 깜빡이는 "Click to Start" 텍스트 애니메이션 추가 (완료)
- 인게임 화면 플레이타임 및 텍스트에 그림자(Text Shadow) 효과 추가 (완료)
## 마일스톤 13: 모바일 최적화 및 부가 요소
- 모바일 터치 시 화면 확대/축소 방지 및 favicon 에러 대응 (완료)
