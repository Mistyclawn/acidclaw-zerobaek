# 오토 개발 크론잡 (Cron Job) 설정

이 프로젝트(acidclaw-zerobaek)는 AcidClaw 에이전트가 매일 지정된 시간에 깨어나 자율적으로 코딩을 수행하도록 설정되어 있습니다.
이 파일은 크론잡의 설정 내용을 보존하며, 크론잡이 변경되더라도 이 기준을 동기화하여 유지해야 합니다.

## 크론잡 명세
- **이름 (Name):** `acidclaw-zerobaek-daily-dev`
- **일정 (Schedule):** 매일 04:00 KST (`0 4 * * *` / Asia/Seoul)
- **대상 (Session Target):** `isolated` (메인 세션 방해 없이 독립적으로 실행)
- **알림 (Notify):** `true`
- **보고 채널 (Delivery):** Discord 채널 `#zerobaek-오토개발` (ID: `1480116101768679425`)
- **사고 수준 (Thinking):** `high` (복잡한 코드 작성 및 컨텍스트 파악을 위함)

## 페이로드 (Payload Message)
크론잡이 실행될 때 에이전트에게 전달되는 정확한 프롬프트입니다:

```text
매일 실행되는 오토 개발(제로백) 태스크입니다.
작업 위치: /Volumes/ROGALLY/github/OpenClaw_auto

지시사항:
1. docs/체크리스트.md, docs/일정.md, docs/Git컨벤션.md 를 읽어주세요.
2. 체크리스트에서 미완료된 최상단 항목(1~2개 정도)을 수행할 코드를 작성/수정하세요 (HTML/JS/CSS 기반).
3. 작업 완료 후, 체크리스트.md의 해당 항목을 `[x]`로 마크하고, 일정.md 등 필요한 문서를 최신화하세요.
4. 작업한 내역을 Git컨벤션.md의 규칙에 따라 `git add .`, `git commit -m '<타입>: <설명>'` 으로 커밋하고 `git push`로 원격에 반영하세요.
5. 모든 작업이 끝나면, **반드시 `message` 도구를 사용하여 Discord 채널(#zerobaek-오토개발, ID: 1480116101768679425)로 작업 결과를 전송**하세요. 보고서 최상단에는 최신 커밋의 ID(Short hash)와 커밋 메시지를 명시해야 합니다.
```

**[주의]** 에이전트는 이 파일을 기준으로 크론잡의 목표와 지시사항을 항상 상기하며, 임의로 오토 개발의 본질적 목표를 변경하지 않습니다.
