# 제로백: 데일리 개발 체크리스트

오토 에이전트는 크론잡 실행 시마다 이 체크리스트를 확인하고 최상단 미완료 태스크부터 수행한다.
수행을 마친 태스크는 `[x]`로 변경하고 진척도를 푸시한다.

- [ ] 프로젝트 기본 폴더 구조화 (`src`, `assets` 등 생성)
- [ ] HTML 파일(index.html) 및 메인 스크립트(main.js), 스타일시트(style.css) 생성
- [ ] Canvas 컨텍스트 로딩 및 리퀘스트애니메이션 루프 뼈대 작성
- [ ] 가상 3D 공간을 표현할 '가짜 3D(Fake 3D / Mode 7 style)' 온레일 렌더러 기초 로직 구현
- [ ] 카메라 중심점 기준 Z축 이동 연산 함수화
- [ ] 마우스 클릭 이벤트 바인딩 (좌클릭/우클릭 스텝 판정)
- [ ] 클릭 타이밍에 맞춘 콤보/비트 연산 로직 추가
- [ ] Godot 기획에 맞춘 장애물(수비수 오브젝트) 생성 및 다가오는 애니메이션 처리
- [ ] 화면 가장자리 글로우 효과(가시광선 파티클 느낌) 구현
- [ ] 황금빛 렌더링(노을 분위기) CSS 필터 전환
- [ ] 튜토리얼 챕터 텍스트/모션 삽입
- [ ] 엔딩 씬 연출 (과거의 자아 일러스트 또는 형태 등장)
