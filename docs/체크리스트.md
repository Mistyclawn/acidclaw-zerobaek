# 제로백: 데일리 개발 체크리스트 (HTML/JS/CSS 기반)

**[중요] 본 프로젝트는 Godot 엔진을 사용하지 않습니다. 순수 웹 기술(HTML5 Canvas, Vanilla JS, CSS)로만 구현합니다.**
오토 에이전트는 크론잡 실행 시마다 이 체크리스트를 확인하고 최상단 미완료 태스크를 1~2개 수행합니다.
수행을 마친 태스크는 `[x]`로 변경하고 진척도를 푸시합니다. 예상치 못한 문제 발생 시 '이슈 추적 및 리팩터링' 섹션에 내용을 기록하고 해결합니다.

- [x] `index.html` 뼈대 작성 (Canvas 요소 및 스크립트/스타일 링크 포함)
